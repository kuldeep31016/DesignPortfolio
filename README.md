## ğŸ’» Usage

The website is a static HTML/CSS site, so no build process is required. Simply open the `index.html` file in any modern web browser to view the site.

### Customization

To customize the website:

1. **Colors**: Edit CSS variables in `styles.css`:
   - `--color-primary`: Main accent color (gold)
   - `--color-dark-bg`: Background color
   - `--color-text-light`: Light text color

2. **Content**: Update text content in `index.html`

3. **Floating Cards**: Adjust card positions in `styles.css` under `.floating-card.card-1`, `.card-2`, `.card-3`
